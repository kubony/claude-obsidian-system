# Typography - 타이포그래피 레퍼런스

한국어 프레젠테이션에 최적화된 타이포그래피 시스템.

---

## 폰트 시스템

### 주요 폰트

| 용도 | 폰트명 | 대체 폰트 | 설명 |
|------|--------|----------|------|
| 제목/강조 | Noto Serif KR | 맑은 고딕 | 세리프 폰트로 고급스러움 |
| 본문/설명 | Pretendard | 맑은 고딕 | 가독성 높은 산세리프 |

### 폰트 설치

- **Noto Serif KR**: [Google Fonts](https://fonts.google.com/noto/specimen/Noto+Serif+KR)
- **Pretendard**: [GitHub](https://github.com/orioncactus/pretendard)

> 폰트가 설치되지 않은 시스템에서는 맑은 고딕으로 대체됩니다.

## 한국어 특화 설정

### 자간 (Letter Spacing)

```
-0.02em ~ -0.03em
```

한글은 영문보다 자간을 약간 좁게 설정하면 자연스럽습니다.

### 행간 (Line Height)

```
1.5 ~ 1.7
```

한글은 영문보다 행간을 넉넉히 설정합니다.

### 단어 분리 (Word Break)

```
word-break: keep-all
```

한글 단어가 중간에 끊기지 않도록 설정합니다.

## 컬러 사용

### 텍스트 컬러

| 요소 | 용도 | 팔레트 변수 |
|------|------|-------------|
| 기본 텍스트 | 제목, 본문 | surface_foreground |
| 보조 텍스트 | 캡션, 레이블, 부제목 | muted_foreground |
| 강조 텍스트 | 숫자, 중요 포인트 | accent |
| 다크 배경 위 | 섹션 슬라이드 텍스트 | dark_fg |

### 컬러 대비

- 배경과 텍스트의 명암비는 최소 4.5:1 유지
- 강조색은 아껴서 사용 (한 슬라이드에 1-2개)

## 모범 사례

### Good ✓

```
# 제목: 단정형 문장으로 메시지 전달
"AI가 비즈니스를 바꾸고 있습니다"

# 본문: 짧고 명확하게
• 핵심 기능 A
• 핵심 기능 B
• 핵심 기능 C
```

### Bad ✗

```
# 제목: 설명형은 피함
"AI의 비즈니스 적용과 그 영향에 대한 분석"

# 본문: 장문은 피함
• 이 기능은 사용자가 더 쉽게 작업을 수행할 수 있도록
  도와주는 혁신적인 기술을 기반으로 만들어졌습니다.
```

## 팁

1. **계층 구조 명확히**
   - 제목 → 소제목 → 본문 순으로 크기 차이 확보
   - 최소 8pt 이상 차이 권장

2. **여백 확보**
   - 텍스트 주변 충분한 여백
   - 슬라이드 가장자리 0.75인치 마진

3. **일관성 유지**
   - 같은 레벨의 텍스트는 같은 스타일
   - 팔레트와 폰트 혼용 금지

4. **가독성 우선**
   - 배경과 대비되는 색상
   - 작은 글씨는 최소화
   - 한 줄에 40자 이내 권장
